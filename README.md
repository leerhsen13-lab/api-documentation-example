# Портфолио: Документирование REST API

В данном репозитории представлен пример моей работы в качестве IT-технического писателя по документированию интеграционных сервисов (REST API).

### Что внутри:
* [card_api_spec.md](./card_api_spec.md) - Спецификация API-метода `GET /api/v1/cards/balance` - Запрос баланса и истории транзакций по карте клиента.
* [card_block_api.md](./card_block_api.md) - Спецификация API-метода `POST /api/v1/cards/block` - Запрос блокировки банковской карты клиента.

### Использованный стек технологий:
* **Формат данных:** JSON
* **Документирование:** Markdown (Docs-as-Code)
* **Инструменты тестирования:** Postman
* **Архитектура:** REST API
